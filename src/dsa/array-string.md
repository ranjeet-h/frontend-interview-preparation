# Array & String Problems

> **Canonical chapter.** Object/application problems → [Object & App Problems](object-app.md). JS coding fundamentals → [JS Coding](../javascript/coding.md).

This chapter covers algorithm problems focused on arrays and strings — the most common interview problem types.

**Problem categories:**
- Array-based problems: two-sum, sorting variants, sliding window, frequency counts
- String manipulation: character frequency, palindrome, anagram, substring search

**Study approach:**
1. Define the state clearly before writing code
2. State the loop invariant
3. Start with brute force O(n²) — then optimize
4. Always analyze time and space complexity
