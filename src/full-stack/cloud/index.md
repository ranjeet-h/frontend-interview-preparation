# Cloud & DevOps

> Planned track for AWS, Azure, Google Cloud, deployment, and operations.

## Planned chapters

| Chapter | Topics |
|---|---|
| AWS Core Services | IAM, EC2, S3, RDS, Lambda, SQS, CloudFront |
| Azure Core Services | Entra ID, VMs, Storage, Functions, SQL, networking |
| Google Cloud Core Services | IAM, Compute Engine, Cloud Storage, Cloud Run, Pub/Sub |
| Delivery & Operations | CI/CD, containers, infrastructure as code, secrets, observability |
| Reliability | Scaling, failover, backups, disaster recovery, incident response |

## Expansion rule

Keep cloud chapters aligned by capability, not by vendor names alone, so the book can compare
equivalent services cleanly.
