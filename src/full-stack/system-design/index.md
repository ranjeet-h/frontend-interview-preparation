## How navigation works

- Use this page to choose a question family.
- Each chapter page includes a question index that links to every question on that page.
- Every solution section begins with an answer summary and hides the raw source notes below it.

## Diagram support

- Mermaid is used for chapter-level architecture snapshots.
- Existing source diagrams remain in the collapsed original source notes.
- The book is wired for mdBook Excalidraw support, so richer diagrams can be added later without changing the chapter layout.
