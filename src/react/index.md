# React

React ties together state, rendering, and reusable UI architecture. Study it by understanding how data flows, when components re-render, and how hooks shape that flow.

## Chapters

| Chapter | Focus |
|---|---|
| [Theory & Core Concepts](theory.md) | Virtual DOM, hooks, state management, component architecture, and performance |
| [Coding Challenges](coding.md) | Custom hooks, component implementations, memoization, and re-render behavior |

## Key React concepts summary

- **Virtual DOM** — understand how React batches work and compares trees.
- **Hooks** — know when `useState`, `useEffect`, `useMemo`, and `useCallback` run.
- **State management** — know when to use local state, Context, or a store.
- **Performance** — focus on memoization, splitting, and avoiding unnecessary renders.
