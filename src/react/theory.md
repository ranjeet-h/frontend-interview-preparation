# React Theory & Core Concepts

> **Canonical chapter for React theory.** React coding challenges → [React Coding](coding.md). JS foundations that React builds on → [JS Theory](../javascript/theory.md).

This chapter is the primary reference for React conceptual knowledge covering:

- **Core React concepts** — Virtual DOM, diffing algorithm, keys, controlled vs uncontrolled components, HOCs, createElement vs cloneElement, lifting state, prop drilling
- **Hooks in-depth** — useState, useEffect (dependency array), useReducer, useMemo, useCallback, useRef, custom hooks, useLayoutEffect
- **State management** — Context API, Redux (Store/Actions/Reducers), Zustand
- **Performance optimization** — React.memo, code splitting, common bottlenecks, profiling
- **Component architecture** — scalable structure, presentational vs container, render props
- **Testing** — Jest, React Testing Library, shallow vs mount, mocking, snapshot testing
- **React 18+ features** — Concurrent Rendering, Server Components, `use` hook, Suspense
- **Advanced hooks & patterns** — useImperativeHandle, Error Boundaries, Portals, StrictMode
- **TypeScript with React** — typing props, state, hooks
- **Rendering strategies** — SSR, CSR, SSG, ISR (canonical reference — also covered in [Web Concepts](../frontend/web-concepts.md))
- **Architectural design** — reusable Data Grid component design example
