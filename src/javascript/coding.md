# JavaScript Coding Problems & Deep Dives

> **Canonical chapter for JS coding problems.** Core theory → [JS Theory](theory.md). Algorithm-focused problems → [DSA](../dsa/index.md).

This chapter contains a comprehensive set of JavaScript coding problems organized by topic. It also includes extended theory deep-dives that complement the [JS Theory](theory.md) chapter with additional examples and interview-level depth.

**Topics in this chapter:**

- Variable declarations, hoisting, scope, closures
- Asynchronous JavaScript (Promises, async/await, event loop)
- ES6+ features (destructuring, modules, generators, spread/rest)
- Object-oriented patterns (classes, inheritance, mixins)
- DOM manipulation and events
- Error handling patterns
- Data types and coercion edge cases
- Performance optimization implementations
- Advanced patterns (currying, composition, memoization, event emitters)
- Memory management
- Array operations — map, filter, reduce, flatten, sort implementations
- String manipulation problems
- Object handling utilities
- Array method polyfills (map, filter, reduce, forEach, flat, includes, find, findIndex, some, every)
- Function polyfills (bind, call, apply)
- Promise polyfills (Promise, Promise.all, Promise.race, Promise.allSettled, Promise.any)
