# JavaScript Theory & Foundations

> **Canonical chapter for JS theory.** For coding problems and implementations → [JS Coding](coding.md). For DSA → [DSA](../dsa/index.md).

This chapter is the primary reference for JavaScript theoretical concepts. It covers:

- **Prototypes** — Array, String, Object prototypes and their most important methods
- **Looping patterns** — all loop forms over arrays, strings, objects, and nested structures
- **Core concepts** — hoisting, closures, `this`, scope, IIFE, event delegation
- **Asynchronous JavaScript** — event loop, Promises, async/await, microtask vs macrotask
- **ES6+ features** — arrow functions, destructuring, template literals, modules, Set/Map, generators
- **Data structures & algorithms** — reverse string, deep copy, debounce, throttle, flatten, two sum
- **Browser & DOM** — DOM, window vs document, Web Storage, CORS, Critical Rendering Path
- **Performance & memory** — memory leaks, memoization, tree shaking
- **Design patterns** — Module, Observer, Factory patterns in JavaScript
- **Design patterns in JavaScript** (canonical JS patterns reference)
- **Advanced topics** — TypeScript fundamentals, build tools, testing, concurrency

> **Note on overlap:** The [JS Coding](coding.md) chapter also contains extensive theory sections covering many of the same topics with different examples and depth. Use both for full coverage.
