# Polyfills

> **Unique content.** For implementation challenges beyond polyfills → [DSA Implementations](../dsa/implementations.md).
> Canonical theory on Promises → [Core Concepts](core-concepts.md).

Write polyfills from scratch to demonstrate deep understanding of the underlying contract.

**Array method polyfills:** `map`, `filter`, `reduce`, `forEach`, `flat`, `includes`, `find`,
`findIndex`, `some`, `every`

**Function method polyfills:** `bind`, `call`, `apply`

**Promise & async polyfills:** `Promise`, `Promise.all`, `Promise.race`, `Promise.allSettled`,
`Promise.any`, `fetch` (via XHR), `async/await` (via generators), `setTimeout`, `setInterval`

**Other polyfills:** `Object.create`, `requestAnimationFrame`, `localStorage` (via cookies)
